top-scalatra-web
=============

Scalatra web using Maven
-------------------------

**Uses JDK7**

Note that this project will not compile with JDK 6.

```sh
~> mvn jetty:run
```
Point your browser to [http://localhost:8080/top-scalatra-web](http://localhost:8080/top-scalatra-web)
